HTML

  • Head elements contain Metadata that is not shown to the user
  • body elements are what is shown to the user on the webpage
  • HTML is the frame to the website, like a house
  • H1 is the largest font, while H6 is the smallest
  • ul=unordered list with every child text having the LI tag

CSS

  • Margin is the space outside of the elements
  • Padding is the space around the element on the inside of the element

Git

  • git status checks what branch you are in
  • git checkout (branch name) moves you to that branch
  • git checkout -b (branch name) creates new branch and moves you to the branch you created
  • git add -A commits all changes in the working branch
  • git commit -m "message" commits our changes to the branch and adds a message
  • ALWAYS run the git pull origin main command after commiting to insure the branch is up to date before pushing

JavaScript

  • Variable is used to contain any data
  • Control flow is the order in which code is executed
  • Double check spacing as it can cause errors
  • use function (name) to be able to call it back later if conditions are met
