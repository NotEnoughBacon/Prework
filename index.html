Git

  • git status checks what branch you are in
  • git checkout (branch name) moves you to that branch
  • git checkout -b (branch name) creates new branch and moves you to the branch you created
  • git add -A commits all changes in the working branch
  • git commit -m "message" commits our changes to the branch and adds a message
  • ALWAYS run the git pull origin main command after commiting to insure the branch is up to date before pushing
